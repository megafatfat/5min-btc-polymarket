# 5min BTC Polymarket Skill

This repository contains the OpenClaw skill for running and monitoring BTC 5-minute threshold-based trading flows on Polymarket.

## Structure
- `SKILL.md` — skill definition and usage
- `config/` — strategy profiles
- `scripts/` — execution and watchdog scripts
- `examples/` — sample command snippets

## Quick start
1. Review `SKILL.md`
2. Check profile in `config/btc_5m_profiles.yaml`
3. Run scripts from `scripts/` as described in the skill
